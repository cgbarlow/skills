# Skills

Collection of Claude Code skills for enhancing productivity and workflow automation.

## Available Skills

| Skill | Description |
|-------|-------------|
| [agentic-coach](.claude/skills/agentic-coach) | Interactive prompt engineering coach that elevates vague prompts through Socratic dialogue, multiple transformation styles, and guided learning |
| [flow-coach](.claude/skills/flow-coach) | Interactive claude-flow orchestration coach that guides users through swarm topology selection, agent deployment, memory configuration, and SPARC workflows |
| [timesheet](.claude/skills/timesheet) | Interactive timesheet entry — collects work description, date, start/end times via prompts, calculates hours, appends to .campaign/timesheet.md |
| [pdf-to-ebook](.claude/skills/pdf-to-ebook) | Convert a PDF file to ebook formats (epub, mobi, azw3/Kindle) using Calibre |
| [ir3-tax-return](.claude/skills/ir3-tax-return) | Drive the NZ IRD myIR portal to file an individual IR3 income tax return (2026 tax year), prompting for missing data and pausing for RealMe sign-in and final submit |

## Installation

Drop the skill directories into your `~/.claude/skills` directory.
